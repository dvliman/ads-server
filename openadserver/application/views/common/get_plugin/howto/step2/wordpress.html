You don’t have to adjust the plug-in now as we are already made it for you. You can start immediately to display ads on your websites.

  1. Go to the “Appearance → Widgets” interface
  2. From the list of widget drag available widget ”<%SITE_NAME%> Ads” into Widget Area.
  3. In the opened form you need to select from the list a channel and color palette for this channel.
  4. Save the changes by clicking on the “Save” button.

That’s it, now the ads of selected channel will be displayed in created widget. You can add unlimited number of widget for your blog, just repeat previous stages.
