You don’t have to adjust the component now as we are already made it for you. You can start immediately to display ads on your websites.

  1. Go to the “Components → <%SITE_NAME%> Ads” interface
  2. Click on the “Create New Channel” button
  3. In the opened form you need to select from the list a channel and color palette for this channel.
  4. Save the changes by clicking on the “Save” button. The system will automatically create new module in the Joomle system with selected settings.
  5. Adjust module display of created components on your manner.

That’s it, now the ads of selected channel will be displayed in created module. You can add unlimited number of modules for your Joomla system, just repeat previous stages.
