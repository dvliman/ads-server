You don’t have to adjust the module now as we are already made it for you. You can start immediately to display ads on your websites.

  1. In the interface go to “Site Configuration → <%SITE_NAME%> Ads”. In the top select “Add New”.
  2. In the opened form you need to select from the list a channel and color palette for this channel.
  3. Save the changes by clicking on the “Save” button.
  4. After saving the page “Site Bulding → Blocks” is automatically opened. In the list of turn off blocks find the one that has the name of selected by you channel.
  5. Set the region for this block anв click on the “Save blocks” button.
  6. Optional you can adjust output characteristics for this block, by clicking on the “configure” link next to the block.

That’s it, now the ads of selected channel will be displayed in created block. You can add unlimited number of blocks for your website, just repeat previous stages.
