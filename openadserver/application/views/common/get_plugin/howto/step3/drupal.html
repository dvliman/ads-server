Sometimes it is necessity to change settings of the module. Such situation can arise, for instance if: the address of ad server was changed, you have changed api key etc. To avoid reinstallation of the module each time you have made the settings interface.

  1. Go to”Site Configuration → <%SITE_NAME%> Ads”. At the top select “Settings”.
  2. In the opened interface you can inout new appikey, address of ad server or website id.
  3. Save the changes.
  4. Refresh the page.

In case you have entered wrong setting you will see a notification message with a code and error description. You won’t be able to add new channels until the error in the settings will be removed.
