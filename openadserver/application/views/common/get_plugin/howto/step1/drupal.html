1. Upload installation package by clicking on the “Download” button in the bottom screen area.
  2. Unpack the archive into the directory sites/all/modules/ of your website.
  3. Now you need to activate the module, go to the administrative interface of your website.
  4. In the left menu select “Site Bulding → Modules” item.
  5. In the list of available Plug-ins find the block ” <%SITE_NAME%> Ads”. Mark ” <%SITE_NAME%> Core” and “Managed Ads”.
  6. Save the changes.

That’s all the module is successfully installed and activated. Now you can move to adjustment and display of ads.
