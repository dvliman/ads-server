1. Upload installation package by clicking on the “Download” button in the bottom screen area.
  2. In the archive you will find two files “module.zip” and “component.zip”. These are installation packages of component and module. Unpack these files in any suitable for you place.
  3. To install the packages into your CMS system go to administrative panels of Joomla and select “Extensions → Install/Uninstall” menu.
  4. Install the module, for this purpose in the appeared interface press “Browse” button, in the opened window enter the path for the file “module.zip”.
  5. Click on the “Upload File & Install” button
  6. In the same way install the component

That’s all the component and module successfully installed in the system. Now you can move to adjustment and display of ads.
